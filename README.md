## サービス概要
大切な人の誕生日や記念日を、手紙や花束でお祝いするWebアプリケーションです。
花言葉をもとに花束を作成したり、複数人からの手紙を一つのテーブル上にまとめて閲覧することができます。
デジタルでありながら、実際に贈り物を受け取るような体験を提供します。

## このサービスへの思い・作りたい理由
誕生日や記念日に、手紙や花束をもらったときの嬉しさや"大切にされている感覚"は、時間が経っても心に残るものだと思っています。

一方でメッセージアプリやSNSでは、時間が経つとメッセージがタイムライン上で流れてしまい、後から振り返りにくいと感じることがあります。
そこで、誰かの誕生日や記念日のために送った言葉を「ひとつの場所」に残し、何度でも見返せる形にしたいと考え、このアプリを制作したいと思いました。

また、花束を作るときは「花言葉」から選ぶことができ、デジタルであっても"自分のことを考えて選んでくれた"と相手に思ってもらえるような、温度のある体験を目指します。

## ユーザー層について
- 誕生日や記念日を大切にしたい人
- 友人や家族への気持ちを、言葉や形で思い出に残したい人
- サプライズや演出を考えるのが好きな人

## サービスの利用イメージ
お祝いするユーザー
1. 代表者が、お祝いしたい人用のお祝いテンプレを作成する
2. 祝う相手の名前や誕生日・記念日を設定する
3. 投稿用リンクを関係者に共有
4. 関係者はリンクから手紙を投稿【名前】【本文】
5. 代表者が「募集締切」する
6. 代表者が花を検索【花名】【花言葉】【季節】【お祝い内容（例: 誕生日）】
7. 花束の花を決定する
8. 代表者がお祝いするユーザへ共有用URLを送信する

お祝いされるユーザー
1. 閲覧用リンクからアプリを開く
2. 部屋の画面が表示される
3. 部屋の画面内のテーブルをクリック
4. 手紙や花束が置かれた画面が表示される
5. それぞれをクリックし、内容を閲覧する

## ユーザーの獲得について
- SNS（X / Instagram 等）でのシェア
- ポートフォリオやnoteでの紹介

## サービスの差別化ポイント・推しポイント
一般的なメッセージアプリや寄せ書きサービスは、テキスト中心で一覧表示されるものが多いですが、
本サービスでは、自室のテーブルの上に置かれた手紙や花束をクリックして見るという動的な体験を重視しています。
花言葉をもとに花束を作成できる点や、場所を切り替えて思い出のシーンを表現できる点が、差別化ポイントです。
お祝いの気持ちを「読む」だけでなく、「体験する」ことを推しポイントとしています。

## 機能候補
MVPリリース時に作っていたい機能
 ■お祝いする側
- 祝う相手の【名前】【日付】を設定して花束を作成
- 花名・花言葉から花を検索
- 花の説明文
- 選んだ花を花束として構成、一覧表示
- 手紙を作成できる入力フォーム
- 共有用URLの発行

 ■お祝いされる側
- 共有用URLを開いた時の画面（部屋画面～テーブル画面）
- 手紙一覧から、誰からの手紙かを選んで閲覧
- 花束をクリックすると【花名】【花言葉】が表示される
- 作成したお祝いテンプレ・花束・手紙を保存し、後から再閲覧できる

本リリースまでに作っていたい機能
- ユーザー登録・ログイン機能
- 開封演出（募集締切後に初めて全文が見える）
- 手紙の投稿締切日時設定
- 手紙に写真を添付できる機能
- 背景パターンの追加
- テーブル上のアイテムの追加（ケーキ、プレゼントBOX等）
- BGMの設定
