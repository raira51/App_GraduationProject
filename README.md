## サービス概要
大切な人の誕生日や記念日を、手紙や花束でお祝いするWebアプリケーションです。
花言葉をもとに花束を作成したり、複数人からの手紙を一つのテーブル上にまとめて閲覧することができます。
デジタルでありながら、実際に贈り物を受け取るような体験と、その思い出を後から何度でも振り返れる仕組みを提供します。

## このサービスへの思い・作りたい理由
SNSのメッセージで誕生日や記念日をお祝いする際、
”おめでとう”とギフト券を送る流れが、定型化していることに違和感を覚えました。
同じギフト券を何枚も受け取ったという友人の話を聞き、”あなた”をお祝いしているという特別感がより伝わる方法がないかと考えるようになりました。

また、複数人で一人の友人をお祝いする際には、
幹事役が個別に連絡を取り、メッセージを集めてまとめる必要があり、その負担の大きさを何度も目にしてきました。
お祝いの準備を一人が背負うのではなく、全員が関われる体験にできないかと考えました。

一方で、祝われる側としても、SNSのメッセージは時間が経つとメッセージがタイムライン上で流れてしまい、後から振り返ることが難しく、寂しさを感じることがありました。

そこで、お祝いのメッセージをひとつの場所にまとめ、
「デジタル色紙」のようにすることで、
贈る側にとっては“みんなで準備する体験”を、
祝われる側にとっては“記憶として残り、後から振り返れる体験”を
提供できたらいいなと思い、このアプリを制作したいと考えました。

## ユーザー層について
・誕生日や記念日を大切にしたい人
・友人や家族への気持ちを、言葉や形で思い出に残したい人
・サプライズや演出を考えるのが好きな人

## サービスの利用イメージ
お祝いするユーザー
1. 代表者が、お祝いしたい人用のお祝いテンプレを作成する
2. 祝う相手の名前や誕生日・記念日を設定する
3. 投稿用リンクを関係者に共有
4. 関係者はリンクから手紙を投稿【名前】【本文】
5. 代表者が「募集締切」する
6. 代表者が花を検索【花名】【花言葉】【季節】【お祝い内容（例: 誕生日）】
7. 花束の花を決定する
8. 代表者が閲覧用の合言葉を設定する
9. 代表者がお祝いするユーザへ共有用URLを送信する

お祝いされるユーザー
1. 閲覧用リンクからアプリを開く
2. 扉の画面が表示される
3. 閲覧用の合言葉を入力欄へいれる
2. 部屋の画面が表示される
3. 部屋の画面内のテーブルをクリックする
4. 手紙や花束が置かれた画面が表示される
5. それぞれをクリックし、内容を閲覧する

## ユーザーの獲得について
・SNS（X / Instagram 等）でのシェア
・ポートフォリオやnoteでの紹介

## サービスの差別化ポイント・推しポイント
一般的なメッセージアプリや寄せ書きサービスは、テキスト中心で一覧表示されるものが多いですが、本サービスでは、自室のテーブルの上に置かれた手紙や花束をクリックして見るという動的な体験を重視しています。
花言葉をもとに花束を作成できる点や、場所を切り替えて思い出のシーンを表現できる点、お祝いの気持ちを「読む」だけでなく、「体験する」ことを推しポイントとしています。

**競合サービスとの差別化**

| 観点 | 本サービス（卒業制作） | 既存サービス（meseca / comeru 等） |
|---|---|---|
| 主な目的 | お祝いの気持ちを「記憶として残す」 | お祝いを「送る体験」を提供する |
| 体験のゴール | 受け取った人が後から何度も振り返れる | 送信・受け取りが完了した時点で完結 |
| 主役 | 祝われる側（受け取った人） | 送る側 |
| 保存の扱い | 本人が「保存する」と選んだものだけ残る | 自動的に履歴として残ることが多い |
| 一覧画面の意味 | 人生の中に残したい「記憶の棚」 | 作成・受信したメッセージの管理 |
| 役割の考え方 | 送る／受け取るを固定しない | 送る側・受け取る側が明確に分かれる |
| ギフトの扱い | 必須ではない（言葉・気持ちが主役） | ギフト連携が中心になることが多い |
| 重視している価値 | 流れない・残る・振り返れる | すぐ送れる・分かりやすい |

また、本サービスでは、受け取ったお祝いを祝われた本人が「保存する」と選択することで、自分の思い出として管理・再閲覧できる点も特徴です。


## 機能候補
MVPリリース時に作っていたい機能
 ■お祝いする側
 ・祝う相手の【名前】【日付】を設定して花束を作成
 ・花名・花言葉から花を検索
 ・花の説明文
 ・選んだ花を花束として構成、一覧表示
 ・手紙を作成できる入力フォーム
 ・共有用URLの発行

 ■お祝いされる側
 ・共有用URLを開いた時の画面（扉、部屋、テーブル画面）
 ・手紙一覧から、誰からの手紙かを選んで閲覧
 ・花束をクリックすると【花名】【花言葉】が表示される
 ・作成したお祝いテンプレ・花束・手紙を保存し、後から再閲覧できる

本リリースまでに作っていたい機能
 ・ユーザー登録・ログイン機能
 ・開封演出（募集締切後に初めて全文が見える）
 ・手紙の投稿締切日時設定
 ・手紙に写真を添付できる機能
 ・画面背景パターンの追加
 ・テーブル上のアイテムの追加（ケーキ、プレゼントBOX、写真立て等）
 ・BGM、効果音（扉を開く音、手紙をめくる音等）の設定
